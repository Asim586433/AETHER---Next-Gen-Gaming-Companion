# ✨ AETHER - Gaming Companion App

A modern desktop app for gamers with auto-recording, live stats, and cloud storage.

## 🚀 Features
- 🎮 Auto-recording on match detection
- 📊 Live K/D/A tracking
- ☁️ Cloud upload simulation
- 🎨 Beautiful glassmorphism UI
- 📱 Responsive design

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/aether-gaming-companion.git

# Go to project folder
cd aether-gaming-companion

# Install dependencies
npm install

# Run the app
npm start
