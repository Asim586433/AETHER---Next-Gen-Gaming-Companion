## 🚀 Overview

**AETHER** is a premium desktop companion application for competitive gamers, featuring automatic recording, cloud storage, and real-time analytics. Built with Electron and designed with a futuristic SaaS aesthetic.

### ✨ Key Features

- 🎮 **Auto-Recording** - Automatically captures gameplay when matches start
- 📊 **Real-Time Analytics** - Track kills, deaths, and assists live
- ☁️ **Cloud Storage** - Automatic upload to the cloud
- 📈 **Career Statistics** - Track your performance over time
- 🎨 **Modern UI** - Glassmorphism design with dark/light mode
- 🔒 **Secure Authentication** - JWT-based user system
- 💡 **Pro Tips** - AI-powered gaming tips
- 📱 **Responsive** - Works on all screen sizes

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Desktop Framework**: Electron.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Cloud Storage**: AWS S3
- **Game Integration**: Overwolf SDK
